# Books / Public Method Materials

This folder is for transferring, organizing, and extracting method materials from public sources such as the "Alchemy of the Soul" public channel, books, manuals, posts, and educational materials.

Use this folder as a staging and analysis area before integrating ideas into the main method files.

---

## Purpose

Store and process method materials so they can be turned into structured knowledge:

- source texts
- method notes
- extracted principles
- transformation models
- remedy/archetype ideas
- consultation logic
- report logic
- course materials

This folder is not for private client data.

---

## Integration Workflow

1. Add source material or summary.
2. Extract key ideas.
3. Identify method category.
4. Decide whether it belongs in:
   - `method/master-method.md`
   - `method/psychodynamics.md`
   - `method/wu-xing.md`
   - `method/dao-resource-scale.md`
   - `homeopathy/`
   - `alchemy/`
   - `consultations/`
   - `products/`
5. Integrate only refined principles into core method files.
6. Keep raw material separate from final method.

## Safety And Boundaries

Do not store:

- private client data
- identifiable client stories without permission
- secrets
- credentials
- medical guarantees
- financial guarantees

When using public material, keep attribution/source links when relevant.

Do not copy large copyrighted texts unless they are your own material or you have permission. Prefer summaries, excerpts, and extracted principles.
