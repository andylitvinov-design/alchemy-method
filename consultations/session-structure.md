# Session Structure — Alchemy Method

This file defines the basic structure of a consultation/session in Andrey Li's alchemy method.

The session integrates systemic constellations, Dao Resource diagnosis, Wu Xing, homeopathic resonance, neurodiagnostics, psychodynamics, and practical support.

This is a method framework. It does not replace medical, psychological, psychiatric, legal, or financial care.

---

## 1. Core Session Formula

request → field analysis → mechanism → level/resource → remedy/support → action → tracking

A strong session should answer:

- What is the client asking for?
- What is actually happening in the system?
- What is the hidden mechanism?
- What Dao level and Wu Xing resource state are active?
- What remedy/support resonates now?
- What can the client's system safely hold?
- What needs to be done during the next cycle?

## 3. Four-Step Work Model

This model comes from the systemic homeopathy manual.

### Step 1 — Constellation / Situation Analysis

The client describes the desired outcome and draws the scheme:

`I — goal — problem`

Field/constellation analysis shows:

- motivations in the system
- blocks
- cause of difficulties
- solution vector
- hidden mechanism

Session questions:

- What do you want?
- What blocks movement?
- What does the field show as the source?
- What is the first solution movement?

---

### Step 2 — Consultation / Remedy Selection

Homeopathy clarifies:

- depth of the problem
- level of the problem: body / emotions / psyche / deeper symbolic layer
- remedy resonance
- potency/intensity range
- what can be supported now

Typical output:

- 2–3 remedies or support tools for the next cycle
- what each remedy supports
- what to track

Important:
Remedies are selected according to current state, not only textbook description.

---

### Step 3 — Correction / Support

Support options:

1. Client obtains and uses remedies with appropriate guidance.
2. Remote symbolic/resource support through photo/field method, when this is part of the agreed format.
3. Additional practices: body, breath, meditation, nutrition, oils, Bach essences, artifacts.

Tracking period:
Usually 1 week for first changes and remedy relevance check.

Goal:
State shift, stabilization, and movement toward the next level.

---

### Step 4 — Extended Support

After initial change, the practitioner checks:

- what changed
- what stabilized
- what no longer resonates
- what new remedy/support is needed
- whether the client moved to another level/state

Additional alchemical tools may be added:

- crystals
- sephirotic mandalas
- artifacts
- places of power
- oils
- symbolic practices

Purpose:
Strengthen, direct, and stabilize the transformation.

## 7. Remedy Selection Logic

Remedy selection follows:

state → mechanism → Dao level → Wu Xing → remedy resonance → potency/intensity → tracking

Check:

- Does the remedy resonate now?
- What layer does it affect?
- Can the system hold it?
- What potency/intensity is safe?
- What changes should be tracked?
- When should the remedy be reviewed?

---

## 8. Dynamic Remedy Review

Remedy relevance changes over time.

A remedy can be relevant for:

- one day
- one week
- one stage
- one transition

After the state changes, another remedy may become relevant.

Therefore each support cycle must include a review.

Default review rhythm:

- after 1 week for remedy effect
- after 1 month for Dao level movement

## 11. Safety Boundaries

Do not present the session as medical diagnosis or guaranteed treatment.

Use safe language:

- may support
- can help identify
- resonance suggests
- needs verification
- track response

If the client has serious medical, neurological, psychiatric, or emergency symptoms, recommend appropriate professional care.
